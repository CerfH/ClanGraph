ClanGraph (家族智慧图谱)
ClanGraph 是一款基于 Flutter 开发的高级家族关系管理系统。它致力于通过数字化手段，解决现代原子化社会中年轻人对复杂亲属关系“认知断层”的问题，未来将融合 AI 视觉识别与 LLM 家族智慧传承功能。

🌟 项目愿景
认知重构：通过直观的树状/图状结构，清晰呈现家族血缘脉络。

情感链接：不仅记录姓名，更记录往来事务与家族故事。

科技融合：预留 AI 接口，未来支持人脸识别亲戚、方言智慧助手及智能穿戴设备接入。

🚀 当前核心功能
动态图谱：以“我”为核心，实现父辈与子辈的无限节点生长。

智慧录入：全中文交互，支持姓名、称呼、个人特征及备注的详细记录。

数据持久化：本地加密存储，确保你的家族数据不会随页面刷新而丢失。

极客 UI：采用深色模式（Dark Mode）与科技蓝配色，符合顶尖审美。

🛠️ 技术栈
Language: Dart (Flutter 3.41+)

Storage: shared_preferences (本地轻量级存储)

Architecture: 声明式 UI 与模块化数据模型

📦 快速开始
1. 环境准备
确保你的开发环境已通过 flutter doctor 检查。

macOS 15+ / Windows 11

Flutter SDK (Stable channel)

Xcode (用于 iOS 模拟器) 或 Chrome (用于 Web 预览)

2. 获取项目
Bash
git clone [你的仓库地址]
cd clangraph
3. 安装依赖
Bash
flutter pub get
4. 启动预览
在浏览器中查看效果：

Bash
flutter run -d chrome --web-renderer html
